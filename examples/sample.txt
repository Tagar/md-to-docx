## Overview

This is a **sample** input demonstrating every feature supported by `to_docx.py`. Use it as a reference for the syntax you can throw at the converter.

The first non-empty line (above) becomes the document *title* unless you pass `--title`.

## Headings

# H1 via markdown
## H2 via markdown
### H3 via markdown
#### H4 via markdown

ALL CAPS BECOMES H1

1. Short Title Case Becomes H2

## Lists

Numbered list:

1. First item with **bold** and *italic*.
2. Second item with `inline code` and a [link](https://example.com).
3. Third item with ~~strikethrough~~.

Bullets:

- Top-level bullet
- Another top-level
  - Sub-bullet (indent two spaces)
  - Another sub-bullet
- Back to top level

## Inline formatting

Here is **bold**, __also bold__, *italic*, _also italic_, `code`, ~~strike~~, and a [hyperlink](https://anthropic.com).

Inline formatting works in **headings**, list items, and table cells.

Stray <span>html tags</span> get <!-- comments and all --> stripped.
